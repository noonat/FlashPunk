//------------DESCRIPTION------------//
///////////////////////////////////////

This example shows how to generate
levels in Flash Punk with dynamically
created tiles. All of the tiles are
placed in game using the scripts and
functions provided.

The level is generated with Ogmo Editor,
and the project source file is in the
assets folder.

You are free to use this code however
you like, with or without credit,
assuming this readme is kept with all
distributions of the contents of this
example.

You do not have the right to redistribute,
or use the graphics (located in the assets
folder) without my given permission.

This readmy does not apply to the
flashpunk library located in the src/net/ 
folder. For information on that, please 
check out the license.txt in said folder.
